build_tree.py - Output for building the region quadtree

To compile and run the program:

python3 build_tree.py <input.txt>

---------------------------------------------------------------------------------------

intersection.py - Output for region intersection of two quadtrees

To compile and run the program: 

python3 intersection.py <input_1.txt> <input_2.txt>

---------------------------------------------------------------------------------------

Input files 

4x4_1.txt - Homework 2, Tree Generation Example 1 
4x4_2.txt - Homework 2, Tree Generation Example 2
8x8_0.txt - Homework 2, Input File Example

8x8_1.txt - Text file for testing
8x8_2.txt - Text file for testing

----------------------------------------------------------------------------------------

(Note: These programs contain the f-strings format which requires Python 3.6 or above.)
